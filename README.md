## 🏛️ 什么是三省六部虾群？

**三省六部制**是中国古代最成熟的官僚体系，兴盛于大唐盛世。这套体系让一个庞大的帝国能够高效运转，如今我们将其智慧应用到 AI Agent 协作中：

```

│                    中书省（决策层）                       │
│              宰相(8号) - 任务下达、资源协调               │
└────────────────────────────┬────────────────────────────┘
                             │
┌────────────────────────────┼────────────────────────────┐
│                    门下省（审核层）                       │
│          工作流优化师(10号) - 规范校验、质量把控          │
└────────────────────────────┬────────────────────────────┘
                             │
┌────────────────────────────┼────────────────────────────┐
│                    尚书省（执行层）                       │
│  ┌────────┬────────┬────────┬────────┬────────┬────────┐│
│  │  吏部  │  户部  │  礼部  │  兵部  │  刑部  │  工部  ││
│  │  1号   │  9号   │  2号   │  4号   │  7号   │ 3+5号  ││
```

## 📖 快速开始

### 1. 安装技能
```bash
cd ~/.openclaw/skills/
git clone https://github.com/jiebao360/openclaw-san-sheng-liu-bu-lobster.git
```

### 2. 配置 OpenID
编辑 `config/lobster-team.json`，填入各龙虾的 OpenID（参考上面表格）

### 3. 下达第一个任务
```
【任务编号】TASK-20260427-001
【任务主题】草本面膜粉内容营销方案
【任务类型】内容创作
【执行龙虾】龙虾 2 号

【具体要求】
1. 痛点开场：用户最关心的3个肌肤问题
2. 解决方案：草本配方优势说明
3. 功能亮点：突出3个核心成分功效
4. 使用场景：日常护肤3个场景
5. 行动号召：引导到直播间

【字数要求】1500字
【截止时间】20分钟内
```

### 4. 使用生命周期命令
```
/spec 制定内容营销方案规格
/plan 将方案分解为3个子任务
/build 执行文案创作
/test 检查内容质量
/review 复核最终产出
/ship 发布交付，归档知识库
```
